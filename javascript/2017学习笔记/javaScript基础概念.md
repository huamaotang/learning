# javaScript基础概念
## 介绍
```
1、javaScript是一种运行在浏览器的解释型编程语言
2、只有javaScript能跨平台、跨浏览器驱动网页与用户交互
3、网景开发了JavaScript
4、ECMA定制了JavaScript语言标准
```

## 概念
```
1、计算机程序是通过操作值来运行的，能够表示并操作的值的类型称为数据类型
2、javascript允许三种基本的数据类型－－数字、文本字符和布尔值，
	同时还支持两种小数据类型null（空）和undefined（未定义），他们各自只定义了一个值。
	除此之外，还支持符合数据类型－－对象（object）。
3、对象有两种：
    1）已命名的值的无序集合
    2）有编号的值的有序集合，即数组
4、从根本上说，对象和数组属于同一种数据类型，但是它们的行为完全不同，可看作是两种不同类型
5、特殊的对象－－函数（function），函数具有可执行代码的对象，可以通过调用函数来执行某些操作。
6、javascript语言的核心还定义了一些专用的对象。这些对象表示的不是新的数据类型，而是对象的新的类（class）。Date类定义的是表示日期的对象，RegExp类定义的是表示正则表达式的对象，Error类定义的是表示JavaScript程序中发生的语法错误和运行时错误的对象。
```
