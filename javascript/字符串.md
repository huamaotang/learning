
## 字符串
JavaScript的字符串就是用''或""括起来的字符表示。

转义字符\可以转义很多字符，比如\n表示换行，\t表示制表符，字符\本身也要转义，所以\\表示的字符就是\。

```
<script type="text/javascript">
	'use strict';
	var str = 'I\'m \"OK\"!';;
	var s_1 = `
	my 
	\tname
	is\n
	tanghuamao
	`;
	console.log(str);//I'm "OK"!
	console.log(s_1);
	/*
	my 
		name
	is

	tanghuamao*/
</script>
```
